Bloco 1: Fundamentos da Lógica de Programação

Lógica de programação é a base sobre a qual todo o desenvolvimento de software é construído. Antes de aprender qualquer linguagem, é essencial dominar os conceitos que ajudam você a organizar o raciocínio e transformar ideias em soluções automatizadas.

O que é um Algoritmo?

Um algoritmo é uma sequência finita e ordenada de passos para resolver um problema. Assim como uma receita de bolo, ele deve ser claro, preciso e executável.


// Exemplo simples de algoritmo em pseudocódigo:
INÍCIO
  LER número
  SE número % 2 == 0 ENTÃO
    ESCREVER "É par"
  SENÃO
    ESCREVER "É ímpar"
FIM

Tipos de Lógica

  • Lógica Sequencial: Instruções executadas uma após a outra, na ordem.
  • Lógica Condicional: Permite decisões baseadas em condições (ex: IF/ELSE).
  • Lógica Repetitiva: Permite repetir ações (ex: WHILE, FOR).

Boas Práticas para Iniciantes

  • Comece sempre com um rascunho ou fluxograma.
  • Use nomes claros para variáveis e funções.
  • Evite repetir código desnecessariamente (princípio DRY: Don't Repeat Yourself).
  • Teste seu algoritmo com diferentes entradas para ver se cobre todos os casos.

Fluxogramas

Fluxogramas ajudam a visualizar a lógica antes de programar. Cada forma representa uma ação:

  • Retângulo: uma ação ou comando.
  • Losango: uma decisão (sim/não).
  • Oval: início ou fim.

Exemplo Visual (Mental)

Imagine um algoritmo para fazer um café:

  1. Abrir o armário
  2. Pegar o pó de café
  3. Ferver a água
  4. Colocar o pó no filtro
  5. Despejar a água quente
  6. Servir o café

Por que isso importa?

Dominar lógica de programação te prepara para resolver problemas em qualquer linguagem: Python, JavaScript, C++, etc. A linguagem muda, mas a lógica é sempre sua ferramenta número 1.
