Bloco Extra: Aprofundando em JavaScript

JavaScript: Funções, Arrays e Objetos - O Poder de Organização

Com os fundamentos em mente, é hora de aprender a organizar seu código e trabalhar com coleções de dados de forma eficiente.

Funções

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas tornam seu código modular e fácil de manter.

  • Declaração: function nomeDaFuncao(parametros) { /* código */ }
  • Funções Anônimas/Arrow Functions: Sintaxe mais concisa para funções.
  • Escopo: Entender onde as variáveis podem ser acessadas (global vs. local).
  • Retorno: Funções podem retornar valores.

Arrays (Listas de Dados)

Arrays são coleções ordenadas de itens. São incrivelmente úteis para gerenciar listas de qualquer coisa.

  • Criação: const lista = [1, "dois", true];
  • Métodos Comuns: .push() (adicionar), .pop() (remover último), .length (tamanho), .map(), .filter(), .reduce() (para operações avançadas).

Objetos (Dados Estruturados)

Objetos são coleções de pares chave-valor, perfeitos para representar entidades complexas (como um usuário, um produto, etc.).

  • Criação: const usuario = { nome: "Ana", idade: 25 };
  • Acesso: usuario.nome ou usuario['idade'].

Manipulação do DOM (Document Object Model)

O DOM é a ponte entre seu JavaScript e o conteúdo HTML da sua página. É através dele que você pode alterar texto, imagens, estilos e muito mais, tornando a página interativa.

Pense no DOM como uma representação em árvore da sua página HTML. Cada elemento (parágrafo, imagem, botão) é um "nó" nessa árvore.

Como Acessar Elementos HTML:

  • document.getElementById('id-do-elemento')
  • document.querySelector('.classe-ou-tag') (mais versátil)
  • document.querySelectorAll('tag') (retorna uma lista de elementos)

Alterando Elementos:

  • Conteúdo: elemento.textContent = "Novo Texto"; ou elemento.innerHTML = "Novo HTML";
  • Atributos: elemento.setAttribute('src', 'nova-imagem.jpg');
  • Estilo: elemento.style.color = 'red';
  • Classes CSS: elemento.classList.add('nova-classe');

Eventos (Reagindo à Interação do Usuário):

O JavaScript permite que você reaja a ações do usuário, como cliques, digitação, movimento do mouse.

  • Adicionar Ouvinte de Evento: elemento.addEventListener('click', function() { /* faça algo */ });
  • Eventos Comuns: click, mouseover, keydown, submit, etc.

Como praticar?

  • Crie uma página com um botão que mude a cor do fundo da página ao ser clicado.
  • Faça uma lista de tarefas (To-Do List) onde você pode adicionar e remover itens.
  • Crie um contador simples que aumente ou diminua ao clicar em botões.

"O DOM é o palco onde seu JavaScript realiza o show da interatividade."
